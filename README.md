# LeetCodeApp

This is a desktop client for one of the amazing sites for code practice - leetcode.com.

It's written in C++. GUI is based on Qt framework.

<h2> Dependencies: </h2>

<li> libcurl </li>
<li> PyQt </li>
<li> QtWidgets5 </li>

<h2> Supported operating systems: </h2>
<li> Windows </li>
<li> Linux </li>
